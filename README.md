# AI Hub Platform

A production-ready React application for discovering and sharing AI projects.

## Features

- Browse AI models, apps, datasets and MCP tools
- Real-time search and filtering
- Dark mode support
- Fully responsive design
- ⚡ Built with Vite, React, TypeScript, TanStack Router, and Zustand

## Tech Stack

- React
- TypeScript
- TanStack Router
- TanStack React-Query
- Zustand
- Tailwind CSS
- Vite

## Getting Started

```bash
pnpm install
pnpm run dev=
```
